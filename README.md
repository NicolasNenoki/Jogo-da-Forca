# Jogo da Forca - Comida

**Jogo da Forca** feito em Python puro, com tema de **comidas**.

## Como jogar

1. O computador sorteia uma palavra secreta (sempre relacionada a comida)
2. Você tem **6 tentativas** para descobrir a palavra
3. Pode chutar:
   - Uma letra por vez
   - Ou arriscar a palavra inteira de uma vez (se acertar → ganha na hora / se errar → perde 2 tentativas!)
4. A cada erro aparece mais um pedacinho do boneco na forca

## Palavras disponíveis

- ARROZ
- FEIJAO
- ALFACE
- SALAME
- MORANGO
- BATATA
- CARNE
- FRANGO
- PIMENTA
- QUEIJO
- PRESUNTO

## Como rodar

- Direto no terminal!
